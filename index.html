Design System

Monochrome, text-first.

A clean, minimal interface system built on ink and whitespace. No accent color, no decorative images — hierarchy comes from type, spacing, and small inline icons. Akana = ak + ana (clarity + source). Fonts are bundled locally; open any component file directly.
