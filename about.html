About

The system's parts.

Akana is small on purpose — ak (clarity) + ana (source). Three foundations do the heavy lifting; components put them to use. Everything is token-driven and offline.

Foundations

Tokens

tokens.css holds the monochrome palette, type scale, spacing, radii, and motion. The single source of truth — components never hardcode values.

Fonts

IBM Plex Sans + IBM Plex Mono (SIL OFL 1.1) — bundled as woff2 in assets/fonts/ with OFL-IBM-Plex.txt. No CDN. See FONTS.md.

Icons

icons.js is an inline SVG set (1px stroke, currentColor). Use <span class="ak-icon" data-icon="name"> — no image files, ever.

Components

Button

Primary, secondary, ghost. Sizes sm/md/lg. Inline or icon-only.

Card

Icon mark + title + body + text action. A calm surface for grouped content.

Input

Label-led fields with leading icon, helper text, and an error state.

Badge

Outlined status markers; solid only when one item must stand out.

Nav

Icon + text navigation. Active state marked by surface, not color.

Modal

Dialog with focus trap, Escape to close, focus restore.

Toggle

On/off switch via role="switch". State by motion.

Table

Sortable data, aria-sort columns. Text, not color.

Checkbox

Multi-select. Ink fill + check mark. Native input.

Radio

Single choice. Ink ring. Native radio group.

Select

Native dropdown with monochrome chevron.

Textarea

Multi-line field. Same focus language as input.

Tabs

In-page panels. Active = ink underline. Arrow keys.

Alert

Feedback without hue — default, strong rail, solid.

Accordion

Disclosure. Chevron + panel; multi-open by default.

Breadcrumb

Wayfinding trail. Current page is ink weight.

Pagination

List nav. Current page = ink border.

Empty state

Nothing to show — icon, copy, actions.

Patterns

Components composed in context.

Principles

Monochrome. One ink ramp. State is expressed through weight and border, never hue.

Text-first. Type and spacing carry hierarchy. Icons assist scanning; they don't decorate.

No images. Zero decorative imagery. Everything is vector, text, or whitespace.

Offline. Fonts and icons ship with the repo. No network calls on render.
